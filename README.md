# Dashboard UI

A React-based dashboard UI matching the JASMIN design system with dark theme.

## Features

- **Dashboard Page**: Real-time credit risk metrics, application statistics, and data source usage
- **Workflow Builder**: Visual workflow creation interface with building blocks and credit engine panel
- **Monitoring Page**: Comprehensive monitoring with charts, tables, and risk distribution

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Pages

- `/` - Dashboard
- `/workflow` - Workflow Builder
- `/monitoring` - Monitoring

## Tech Stack

- React 18
- React Router
- Vite
- CSS3
